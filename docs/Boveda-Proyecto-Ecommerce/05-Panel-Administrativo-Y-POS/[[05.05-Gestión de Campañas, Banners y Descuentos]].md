# [[05.05-Gestión de Campañas, Banners y Descuentos]]

## 1. Visión General

Este módulo dota al administrador (Store Admin) de un sistema de gestión de contenidos (CMS) enfocado en marketing. Permite modificar el aspecto visual del frontend y aplicar reglas de negocio temporales (descuentos, promociones) sin necesidad de tocar una sola línea de código ni depender de un desarrollador.

## 2. Componentes Visuales Administrables

El sistema permite configurar diferentes "espacios publicitarios" dentro de la tienda:

* **Hero Banners (Carrusel Principal):**
* **Gestión:** Carga de imágenes optimizadas para escritorio y móvil.
* **Configuración:** Título superpuesto, subtítulo, botón de llamada a la acción (CTA) y URL de destino (ej. `/coleccion/one-piece`).
* **Orden:** Capacidad de arrastrar y soltar (drag & drop) para ordenar qué banner aparece primero.


* **Top Bar (Cinta de Anuncios):**
* **Ubicación:** Franja delgada en la parte superior absoluta de la web.
* **Gestión:** Texto editable, selector de color de fondo (HEX) y color de texto. Opción para incluir un enlace o hacerlo estático.


* **Pop-ups Promocionales:**
* **Comportamiento:** Ventana emergente controlada por cookies (para no molestar al usuario en cada recarga). Configurable para aparecer al los 5 segundos de visita o al intentar salir de la página (Exit Intent).


* **Badges (Etiquetas de Producto):**
* Superposiciones automáticas sobre la foto del producto (ej. "30% OFF", "Preventa", "Sold Out").



## 3. Lógica de Programación (Scheduling)

Cada campaña o elemento visual está gobernado por una máquina de estados de tiempo:

* **Borrador (Draft):** Se está diseñando, no es visible para los clientes.
* **Permanente (Always-On):** Activo indefinidamente hasta que el admin decida apagarlo (estado: inactivo).
* **Programado (Time-Bound):**
* Requiere `validFrom` (Fecha/Hora de inicio) y `validUntil` (Fecha/Hora de fin).
* *Ejemplo de uso:* Configurar un banner de "Cyber Days" el martes para que se active automáticamente el viernes a las 00:00 y desaparezca el domingo a las 23:59.



## 4. Motor de Descuentos y Reglas de Precio

Vinculado a lo visual, este motor aplica lógica matemática en tiempo real al carrito de compras:

* **Descuentos Directos (Precios Tachados):**
* Por **Porcentaje (%)** (ej. 15% de descuento).
* Por **Monto Fijo (S/)** (ej. S/ 20 menos).


* **Segmentación de Reglas:**
* **Global:** Aplica a toda la tienda.
* **Por Colección/Categoría:** Aplica solo a figuras de "Dragon Ball".
* **Por Variante:** Aplica descuentos especiales solo a cajas dañadas o figuras "Loose" para limpiar stock.


* **Cupones de Descuento:** Generación de códigos alfanuméricos (ej. `BLUE10`) con límites de uso (ej. "Solo los 50 primeros") o monto mínimo de compra.
