# [[01.04-Librería de Animaciones y Diseño Visual]]

Esta nota documenta las directrices de diseño, animación e identidad visual para **La Tiendita de Blue**, integrando la paleta de colores oficial, tipografías, elementos gráficos interactivos y la adición de tonos pastel para optimizar la experiencia de usuario (UX) en botones de compra y estados de interfaz.

---

## 1. Manual de Identidad Visual (La Tiendita de Blue)

### A. Paleta de Colores Oficiales (Variables CSS en `src/globals.css`)

```css
:root {
  /* Colores Principales de Marca */
  --azul-medio: #4A80BE;       /* Principal (Botones de acción, enlaces, destaques) */
  --azul-oscuro: #142F5C;      /* Contraste / Texto / Encabezados y Navbar */
  --blanco: #FFFFFF;           /* Fondo base limpio */
  --amarillo-estrella: #F5B400;/* Acentos, insignias de preventa, estrellas de reseñas, favoritos */
  --gris-claro: #F2F2F2;       /* Fondos secundarios (tarjetas, sección de detalles) */

  /* Colores Pastel Complementarios (Para suavizar botones y estados) */
  --azul-pastel: #E1F0FF;      /* Fondos de botones secundarios o badges de stock */
  --amarillo-pastel: #FFF5D1;  /* Fondo para notas de preventas o avisos flotantes */
  --rojo-pastel: #FFEAEA;      /* Fondos para alertas, errores o botones de "Eliminar/Cancelar" */
  --verde-pastel: #E2FBE9;     /* Botón de compra rápida o "Añadir al Carrito" (Menos agresivo) */
  --verde-texto: #1E7E34;      /* Texto de confirmación o en stock sobre verde pastel */
}

```

### B. Sistema Tipográfico

Para mantener la jerarquía de lectura exacta de la guía de publicidad:

* **Tipografía Principal (Títulos, Ofertas, Anuncios Clave):** **`Bebas Neue`** (Palo seco condensado, fuerte y con alta presencia visual). Se usará con transformaciones `uppercase` en títulos de productos y secciones principales.
* **Tipografía Secundaria (Textos Generales, Descripciones, Avisos):** **`Montserrat`** (Moderna, geométrica y sumamente legible en cualquier resolución).

## 2. Elementos Visuales y Micro-interacciones (Estilo Homidori + Blue)

### A. Burbujas con Costura (Contenedores Destacados)

* **Uso:** Contenedores de categorías, banners de promoción o destacados en el Home.
* **Implementación:** Bordes redondeados con la propiedad `border-dashed border-2 border-azul-medio` recreando la costura artesanal de la marca, combinado con el personaje **Bluet** asomándose en esquinas clave del sitio para generar cercanía.

### B. Animación de Botones (Librería de Animaciones)

* **Botón de Añadir al Carrito (Verde/Azul Pastel):** Al pasar el cursor (*hover*), transiciona suavemente de un fondo pastel (`bg-azul-pastel` o `bg-verde-pastel`) a un color sólido y vibrante (`bg-azul-medio`), con un ligero efecto de rebote (*spring*) para incentivar el clic.
* **Estrellas de Reseña (Amarillo Estrella):** Animación de escala y destello dorado al pasar la puntuación en la ficha del producto (`[id]/page.tsx`).

### C. Íconos Lineales y Mascotas (Bluet)

* **Íconos de Envío y Procesos:** Dibujos vectoriales sencillos (estilo el avión de papel de la guía de marca) para marcar los tipos de envío (Shalom, Motorizado, Presencial).
* **Mascota (Bluet):** Animación de entrada con un leve balanceo (*bounce-in*) al cargar la pantalla de Checkout exitoso, reforzando el mensaje de "¡Gracias por tu compra!".

---

## 3. Prevención de Errores Visuales (UX Friccionada)

* **Estado de Carga (Spinners):** Mientras se procesa un pago o se sube el comprobante al POS, el botón de compra deshabilitado mostrará una burbuja de costura pequeña dando vueltas lentamente en tono `azul-medio`.
* **Notificaciones de Aviso (Toast Notifications):** Alertas suaves con esquinas muy redondeadas utilizando los colores pastel de fondo con bordes sutiles para dar feedback de éxito sin asustar al usuario con rojos o amarillos de alto contraste.
