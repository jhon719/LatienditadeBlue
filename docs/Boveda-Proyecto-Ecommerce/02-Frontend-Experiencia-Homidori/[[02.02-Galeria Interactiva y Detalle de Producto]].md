# Bloque 3: Lógica de Compra en Grupo (Bundle / Up-selling)

## 1. Flujo de Datos y Lógica de Negocio

Para que la experiencia sea fluida y no afecte el rendimiento, el cálculo de los precios se realiza **completamente en el cliente (Client-side)** en tiempo real, mientras que la validación final del stock y precios congelados se realiza en el servidor al enviar la orden.

### Regla de Descuento Dinámico

Para incentivar que el usuario mantenga todos los productos seleccionados, aplicaremos un descuento escalonado:

* **1 Producto seleccionado:** Sin descuento (Precio regular).
* **2 Productos seleccionados:** **5% de descuento** sobre la suma total de los productos seleccionados.
* **3 o más Productos seleccionados:** **10% de descuento** sobre la suma total de los productos seleccionados.

---

## 2. Componente React: `src/components/products/ProductBundle.tsx`

Este componente de Next.js (Client Component) maneja el estado de selección, renderiza las figuras del set (alojadas en Cloudinary) y despacha la acción de compra masiva al carrito.

```tsx
"use client";

import { useState, useEffect } from "react";
import Image from "next/image";
import { useCartStore } from "@/stores/cart-store";

interface BundleProduct {
  id: string;
  name: string;
  price: number;
  imageUrl: string;
  slug: string;
}

interface ProductBundleProps {
  currentProduct: BundleProduct;
  suggestedProducts: BundleProduct[]; // Máximo 2 sugeridos adicionales
}

export default function ProductBundle({ currentProduct, suggestedProducts }: ProductBundleProps) {
  const addToCart = useCartStore((state) => state.addToCart);
  
  // Unimos todos los productos del bundle (el actual siempre está primero)
  const allBundleProducts = [currentProduct, ...suggestedProducts];
  
  // Estado para rastrear qué productos están marcados (por ID)
  // Por defecto, todos se inicializan marcados para forzar el descuento inicial
  const [selectedIds, setSelectedIds] = useState<string[]>(
    allBundleProducts.map((p) => p.id)
  );

  const [totals, setTotals] = useState({
    original: 0,
    discounted: 0,
    discountPercent: 0,
    savedAmount: 0,
  });

  // Alternar selección de un producto (El producto principal no se puede desmarcar)
  const handleToggle = (id: string) => {
    if (id === currentProduct.id) return; // Obligatorio el producto actual

    setSelectedIds((prev) =>
      prev.includes(id) ? prev.filter((item) => item !== id) : [...prev, id]
    );
  };

  // Recalcular montos cada vez que cambia la selección
  useEffect(() => {
    const selectedProducts = allBundleProducts.filter((p) => selectedIds.includes(p.id));
    
    // Suma base
    const sumOriginal = selectedProducts.reduce((acc, p) => acc + p.price, 0);
    
    // Calcular porcentaje de descuento según cantidad
    let discountPercent = 0;
    if (selectedProducts.length === 2) discountPercent = 5;
    else if (selectedProducts.length >= 3) discountPercent = 10;

    const discountAmount = sumOriginal * (discountPercent / 100);
    const sumDiscounted = sumOriginal - discountAmount;

    setTotals({
      original: sumOriginal,
      discounted: sumDiscounted,
      discountPercent,
      savedAmount: discountAmount,
    });
  }, [selectedIds]);

  // Agregar todos los seleccionados al carrito global (Zustand)
  const handleAddBundleToCart = () => {
    const productsToAdd = allBundleProducts.filter((p) => selectedIds.includes(p.id));
    
    productsToAdd.forEach((product) => {
      addToCart({
        id: product.id,
        name: product.name,
        price: product.price,
        imageUrl: product.imageUrl,
        quantity: 1,
      });
    });

    // Opcional: Disparar modal lateral del carrito indicando éxito
  };

  return (
    <div className="bg-white border-2 border-dashed border-azul-medio rounded-3xl p-6 my-8">
      {/* Título en Bebas Neue */}
      <h3 className="font-bebas text-2xl text-azul-oscuro tracking-wide mb-6 uppercase">
        ⚡ ¡Lleva el set completo y ahorra!
      </h3>

      <div className="flex flex-col lg:flex-row items-center justify-between gap-6">
        {/* Lado Izquierdo: Galería del Set */}
        <div className="flex flex-wrap items-center justify-center gap-4">
          {allBundleProducts.map((product, index) => {
            const isSelected = selectedIds.includes(product.id);
            const isMain = product.id === currentProduct.id;

            return (
              <div key={product.id} className="flex items-center">
                {/* Conector "+" visual */}
                {index > 0 && (
                  <span className="text-2xl font-bold text-azul-medio mx-2">+</span>
                )}

                <div
                  onClick={() => handleToggle(product.id)}
                  className={`relative flex flex-col items-center p-3 rounded-2xl border-2 cursor-pointer transition-all ${
                    isSelected 
                      ? "border-azul-medio bg-azul-pastel/30" 
                      : "border-gray-200 opacity-50 hover:opacity-80"
                  }`}
                >
                  {/* Checkbox circular interactivo */}
                  <div className="absolute top-2 right-2 z-10">
                    <input
                      type="checkbox"
                      checked={isSelected}
                      disabled={isMain}
                      readOnly
                      className="w-4 h-4 rounded-full text-azul-medio focus:ring-azul-medio"
                    />
                  </div>

                  {/* Imagen del Producto (Cloudinary optimizada por Next.js) */}
                  <div className="w-24 h-24 relative mb-2">
                    <Image
                      src={product.imageUrl}
                      alt={product.name}
                      fill
                      className="object-contain rounded-lg"
                    />
                  </div>

                  {/* Detalle en Montserrat */}
                  <p className="font-montserrat text-xs text-center font-bold text-azul-oscuro max-w-[100px] truncate">
                    {product.name}
                  </p>
                  <p className="font-montserrat text-xs text-azul-medio font-semibold mt-1">
                    S/. {product.price.toFixed(2)}
                  </p>

                  {isMain && (
                    <span className="absolute -bottom-2 bg-amarillo-estrella text-azul-oscuro font-montserrat text-[10px] font-extrabold px-2 py-0.5 rounded-full uppercase">
                      Este producto
                    </span>
                  )}
                </div>
              </div>
            );
          })}
        </div>

        {/* Lado Derecho: Caja de Totales y Conversión */}
        <div className="w-full lg:w-72 bg-gris-claro rounded-2xl p-5 border border-gray-200">
          <p className="font-montserrat text-xs text-gray-500 uppercase font-semibold">
            Resumen del Set ({selectedIds.length} productos)
          </p>

          <div className="my-3 space-y-1.5">
            {totals.discountPercent > 0 && (
              <div className="flex justify-between items-center">
                <span className="font-montserrat text-sm text-gray-600">Precio Regular:</span>
                <span className="font-montserrat text-sm line-through text-gray-400">
                  S/. {totals.original.toFixed(2)}
                </span>
              </div>
            )}

            {totals.discountPercent > 0 && (
              <div className="flex justify-between items-center">
                <span className="font-montserrat text-sm text-verde-texto font-bold">Descuento ({totals.discountPercent}%):</span>
                <span className="font-montserrat text-sm text-verde-texto font-bold">
                  - S/. {totals.savedAmount.toFixed(2)}
                </span>
              </div>
            )}

            <div className="flex justify-between items-end pt-2 border-t border-gray-300">
              <span className="font-montserrat text-base font-bold text-azul-oscuro">Total Set:</span>
              <span className="font-bebas text-3xl text-azul-oscuro tracking-wide">
                S/. {totals.discounted.toFixed(2)}
              </span>
            </div>
          </div>

          {/* Botón de Compra del Bundle con efectos pastel de la marca */}
          <button
            onClick={handleAddBundleToCart}
            className="w-full bg-azul-medio hover:bg-azul-oscuro text-white font-montserrat font-bold py-3 px-4 rounded-xl transition-all shadow-md active:scale-95 flex items-center justify-center gap-2"
          >
            🛒 Añadir set al carrito
          </button>
        </div>
      </div>
    </div>
  );
}

```

# Bloque 2: Ficha de Conversión y Botones (`<ProductPurchaseSection/>`)

Este bloque es el motor transaccional de la página de producto. Su objetivo es comunicar con total claridad el estado del inventario (Stock regular vs. Preventa), calcular subtotales de manera dinámica y ofrecer llamadas a la acción (CTAs) sumamente intuitivas y atractivas.

---

## 1. Lógica de Estados y Reactividad

Para asegurar una experiencia de usuario fluida y sin fricciones, el componente maneja tres estados principales en el cliente:

1. **`quantity` (Número):** Controla la cantidad de piezas que el usuario desea comprar. Está limitada por un mínimo de `1` y un máximo equivalente al stock disponible (o un límite comercial de `5` unidades para evitar acaparamiento en preventas de figuras).
2. **`isWishlisted` (Booleano):** Estado local interactivo que gestiona si el producto está guardado en la lista de deseos del usuario, aplicando una micro-animación de escala en el ícono de corazón.
3. **`isSubmitting` (Booleano):** Evita el "doble clic" accidental en los botones de compra bloqueando los CTAs mientras se despacha la acción al carrito de Zustand.

---

## 2. Componente React: `src/components/products/ProductPurchaseSection.tsx`

Este componente utiliza Tailwind CSS para dar un acabado limpio, adaptándose perfectamente a la paleta suave y a la tipografía de la marca (**Bebas Neue** para títulos de impacto y **Montserrat** para textos de lectura).

```tsx
"use client";

import { useState } from "react";
import { useCartStore } from "@/stores/cart-store";
import { Heart, ShoppingBag, CreditCard, Share2, Info } from "lucide-react";

interface ProductPurchaseProps {
  product: {
    id: string;
    name: string;
    brand: string;
    price: number;
    stock: number;
    isPreOrder: boolean;
    expectedDate?: string; // Ej: "Octubre 2026"
    description: string;
    imageUrl: string;
  };
}

export default function ProductPurchaseSection({ product }: ProductPurchaseProps) {
  const addToCart = useCartStore((state) => state.addToCart);
  const [quantity, setQuantity] = useState(1);
  const [isWishlisted, setIsWishlisted] = useState(false);
  const [isSubmitting, setIsSubmitting] = useState(false);

  const maxLimit = product.isPreOrder ? 5 : Math.min(product.stock, 10);
  const subtotal = product.price * quantity;

  const handleIncrement = () => {
    if (quantity < maxLimit) setQuantity((prev) => prev + 1);
  };

  const handleDecrement = () => {
    if (quantity > 1) setQuantity((prev) => prev - 1);
  };

  const handleAddToCart = async () => {
    setIsSubmitting(true);
    // Simulación de retraso sutil para feedback de UI
    await new Promise((resolve) => setTimeout(resolve, 300));
    
    addToCart({
      id: product.id,
      name: product.name,
      price: product.price,
      imageUrl: product.imageUrl,
      quantity: quantity,
    });
    
    setIsSubmitting(false);
  };

  return (
    <div className="flex flex-col gap-5 p-2">
      {/* Cabecera: Marca y Título */}
      <div>
        <span className="font-montserrat text-xs font-bold uppercase tracking-wider text-azul-medio">
          {product.brand}
        </span>
        <h1 className="font-bebas text-3xl md:text-4xl text-azul-oscuro tracking-wide mt-1 leading-none">
          {product.name}
        </h1>
      </div>

      {/* Badge de Stock o Preventa */}
      <div className="flex items-center gap-2">
        {product.isPreOrder ? (
          <div className="flex flex-col gap-1">
            <span className="inline-flex items-center gap-1.5 px-3 py-1 rounded-full text-xs font-bold bg-amarillo-pastel text-azul-oscuro border border-dashed border-amarillo-estrella w-fit">
              <span className="w-2 h-2 rounded-full bg-amarillo-estrella animate-pulse" />
              Preventa Asegurada
            </span>
            {product.expectedDate && (
              <span className="font-montserrat text-xs text-gray-500 flex items-center gap-1">
                <Info size={12} className="text-azul-medio" />
                Llegada estimada: {product.expectedDate}
              </span>
            )}
          </div>
        ) : product.stock > 0 ? (
          <span className="inline-flex items-center gap-1.5 px-3 py-1 rounded-full text-xs font-bold bg-verde-pastel text-verde-texto w-fit">
            <span className="w-2 h-2 rounded-full bg-verde-texto animate-ping" />
            En Stock - Envío Inmediato
          </span>
        ) : (
          <span className="inline-flex items-center gap-1.5 px-3 py-1 rounded-full text-xs font-bold bg-red-100 text-red-600 w-fit">
            Agotado temporalmente
          </span>
        )}
      </div>

      {/* Precio y Desglose */}
      <div className="border-y border-gray-100 py-4 my-1">
        <div className="flex items-baseline gap-2">
          <span className="font-bebas text-4xl text-azul-oscuro tracking-wide">
            S/. {product.price.toFixed(2)}
          </span>
          <span className="font-montserrat text-xs text-gray-400">
            Impuestos incluidos
          </span>
        </div>
        <p className="font-montserrat text-sm text-gray-600 mt-2 leading-relaxed">
          {product.description}
        </p>
      </div>

      {/* Selector de Cantidad */}
      {product.stock > 0 || product.isPreOrder ? (
        <div className="flex items-center gap-4">
          <span className="font-montserrat text-xs font-bold text-azul-oscuro uppercase">
            Cantidad:
          </span>
          <div className="flex items-center border-2 border-azul-pastel rounded-xl overflow-hidden bg-white">
            <button
              onClick={handleDecrement}
              disabled={quantity <= 1}
              className="px-3 py-1.5 text-azul-oscuro hover:bg-azul-pastel/30 transition-colors disabled:opacity-40 font-bold"
            >
              -
            </button>
            <span className="px-4 py-1.5 font-montserrat font-bold text-sm text-azul-oscuro w-10 text-center select-none">
              {quantity}
            </span>
            <button
              onClick={handleIncrement}
              disabled={quantity >= maxLimit}
              className="px-3 py-1.5 text-azul-oscuro hover:bg-azul-pastel/30 transition-colors disabled:opacity-40 font-bold"
            >
              +
            </button>
          </div>
          {/* Subtotal Dinámico */}
          <span className="font-montserrat text-xs text-gray-400 ml-auto">
            Subtotal: <strong className="text-azul-oscuro text-sm">S/. {subtotal.toFixed(2)}</strong>
          </span>
        </div>
      ) : null}

      {/* Botones de Acción (CTAs) */}
      <div className="flex flex-col gap-3 mt-2">
        {product.stock > 0 || product.isPreOrder ? (
          <>
            <div className="grid grid-cols-1 sm:grid-cols-2 gap-3">
              {/* Botón Añadir al Carrito */}
              <button
                onClick={handleAddToCart}
                disabled={isSubmitting}
                className="w-full flex items-center justify-center gap-2 bg-azul-pastel/50 hover:bg-azul-pastel border-2 border-dashed border-azul-medio text-azul-oscuro font-montserrat font-bold py-3.5 px-4 rounded-2xl transition-all active:scale-95 disabled:opacity-50"
              >
                <ShoppingBag size={18} />
                {isSubmitting ? "Agregando..." : "Agregar al Carrito"}
              </button>

              {/* Botón Comprar Ahora */}
              <button
                disabled={isSubmitting}
                className="w-full flex items-center justify-center gap-2 bg-azul-medio hover:bg-azul-oscuro text-white font-montserrat font-bold py-3.5 px-4 rounded-2xl transition-all shadow-md active:scale-95 disabled:opacity-50"
              >
                <CreditCard size={18} />
                Comprar Ahora
              </button>
            </div>

            {/* Acciones Secundarias */}
            <div className="flex items-center justify-start gap-4 mt-1 border-t border-gray-100 pt-3">
              {/* Wishlist Button */}
              <button
                onClick={() => setIsWishlisted(!isWishlisted)}
                className="flex items-center gap-1.5 font-montserrat text-xs text-gray-500 hover:text-red-500 transition-colors group"
              >
                <Heart
                  size={16}
                  className={`transition-all duration-300 ${
                    isWishlisted 
                      ? "fill-red-500 text-red-500 scale-110" 
                      : "text-gray-400 group-hover:scale-105"
                  }`}
                />
                {isWishlisted ? "En tu lista" : "Añadir a deseos"}
              </button>

              {/* Share Button */}
              <button className="flex items-center gap-1.5 font-montserrat text-xs text-gray-500 hover:text-azul-medio transition-colors group">
                <Share2 size={16} className="text-gray-400 group-hover:scale-105 transition-transform" />
                Compartir
              </button>
            </div>
          </>
        ) : (
          <button
            disabled
            className="w-full bg-gray-200 text-gray-400 font-montserrat font-bold py-3.5 px-4 rounded-2xl cursor-not-allowed text-center"
          >
            Temporalmente sin Stock
          </button>
        )}
      </div>
    </div>
  );
}
```

## 3. Detalles de Diseño y Micro-interacciones

* **Feedback Visual de Entrada de Datos:** Al pulsar los botones de más (`+`) o menos (`-`), el contador se actualiza instantáneamente de manera reactiva, recalculando el valor del subtotal a un costado sin alterar la disposición del resto de componentes.
* **Transiciones de Estado de Carga:** El estado `isSubmitting` inhabilita de inmediato ambos botones de compra para evitar compras por duplicado en sistemas lentos y transforma el texto del botón en un estado de espera dinámico.
* **Badge Dinámico de Preventa:** Si el producto está catalogado como preventa, el badge incorpora un pequeño círculo animado con `animate-pulse` que simula "actividad", captando el interés visual hacia la fecha de llegada proyectada..
